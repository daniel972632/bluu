📝 QA Daily Report Tool (일일리포트 작성기)

QA 본부 리포트 작성을 효율적으로 도와주는 웹 기반 도구입니다. 팀원들의 업무 내용을 입력하고 정렬하며, 최종적으로 JSON 파일로 내보내거나 병합할 수 있습니다.

✨ 주요 기능

업무 기록 관리: 이름, 프로젝트별로 금일 또는 차주 업무 내용을 간편하게 추가하고 관리합니다.

자동 정렬: 설정에 따라 이름순 또는 프로젝트순으로 테이블 데이터를 실시간 정렬합니다.

데이터 영속성: 브라우저의 LocalStorage를 사용하여 페이지를 새로고침해도 입력 데이터가 유지됩니다.

JSON 내보내기 및 다운로드: 작성된 데이터를 JSON 형식으로 복사하거나 파일로 다운로드할 수 있습니다.

스마트 파일명: 이름_날짜_구분.json (예: 조현열_260204_일일.json) 형태로 자동 생성됩니다.

팀원 데이터 병합: 팀원들이 내보낸 JSON 데이터를 붙여넣거나 Drag & Drop 하여 하나로 합칠 수 있습니다.

실행 취소 (Undo): 병합 작업이 잘못되었을 경우 이전 상태로 되돌릴 수 있습니다.

Bongo Cat 이스터 에그: 특정 명령어를 입력하면 귀여운 Bongo Cat 애니메이션이 나타납니다.

🚀 사용 방법

1. 업무 추가

이름과 프로젝트를 선택하거나 직접 입력합니다.

업무 내용을 입력하고 Enter 키 또는 추가 버튼을 누릅니다.

추가된 항목을 클릭하면 삭제되며, 해당 내용이 다시 입력창으로 복구되어 수정이 용이합니다.

2. 데이터 관리

JSON 복사: 전체 데이터를 클립보드에 JSON 형식으로 복사합니다.

파일 다운로드: 현재 설정된 이름과 날짜를 기준으로 .json 파일을 생성합니다.

초기화: 모든 데이터를 삭제하고 리스트를 비웁니다.

3. 팀 데이터 병합

다른 팀원의 JSON 데이터를 하단 텍스트 영역에 붙여넣고 병합하기를 누르거나, 파일을 화면 위로 드래그 앤 드롭 하세요.

🛠 기술 스택

Frontend: React (v18)

Styling: Tailwind CSS

Icons & Animation: Inline SVG & CSS Animations

Storage: Browser LocalStorage

Deployment: Single HTML File (No Build Required)

⚙️ 설정 커스터마이징

코드 상단의 TEAM_NAMES 및 PROJECTS 상수를 수정하여 팀의 상황에 맞게 최적화할 수 있습니다.

const TEAM_NAMES = ["김수은", "장원형", "이동준", "이세인", "조현열"];
const PROJECTS = [
    "bluu Cloud", "bluu Point", "bluu Restaurant", ...
];
